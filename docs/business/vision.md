# IPMax — 사업 비전

## 한 문장

> 개인과 기업의 유휴 네트워크 자원을 자산화하여, 필요한 이에게 초 단위로 거래되는 **글로벌 IP 마켓플레이스**.

## 문제 (Problem)

- 기존 VPN·프록시 시장은 **소수 사업자의 IP 풀**에 의존해 가격·품질이 경직됨
- 사용자는 **필요한 지역/품질**을 세분화해 고르기 어려움
- 제공자(회선 소유자)는 **유휴 자원을 수익화할 표준 경로**가 없음
- 차단 환경(DPI, GFW 등)에서 안정 접속이 어려움

## 해결 (Solution)

- **양면 마켓플레이스**: Provider(회선 공급) ↔ Consumer(IP 소비)
- **WireGuard 기반 고속 터널** + **프록시(HTTP/SOCKS5)** + **DPI 우회**
- **초 단위 트래픽 정산**으로 사용량에 정확히 비례하는 과금·수익
- **충전형 크레딧** — 선결제·자동차감으로 운영/회계 단순화

## 타겟 (B2B2C)

| 세그먼트 | Provider | Consumer |
|---|---|---|
| **개인** | 집 회선 유휴 시간대 수익화 | 게임·스트리밍·지역 차단 우회 |
| **소기업(SMB)** | 사무실 다중 회선 활용 | 크롤링·QA·지역 테스트 |
| **기업(Enterprise)** | 데이터센터/지사 회선 재판매 | 지역별 마케팅·광고 검증·보안 테스트 |

## 차별화

1. **초 단위 정산** — 분·시간 단위 경쟁사 대비 낭비 없음
2. **OpenWrt 기반 플러그앤플레이 Provider 키트** — 설치 장벽 최소화
3. **멀티 플랫폼 네이티브 클라이언트** — Windows/Mac/iOS/Android 모두 네이티브 UX
4. **투명한 지역·품질 스코어링** — 가짜 위치·불안정 노드 필터링

## 비즈니스 모델

- 거래 수수료 (Consumer 결제액의 일정 비율)
- 프리미엄 스코어링/우선순위 노출 (Provider 옵션)
- 기업 전용 전용선·SLA (Enterprise 플랜)

## 성공 지표 (예시)

- 월 거래 트래픽 (TB)
- 활성 Provider 수 / 활성 Consumer 수
- 재결제율, 평균 크레딧 충전액
- Endpoint 가동률, 세션 성공률
